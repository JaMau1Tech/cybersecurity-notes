# Templates

This folder contains reusable templates that help standardize documentation across projects, labs, investigations, and write-ups.

Using consistent templates improves organization, readability, and documentation quality.

---

# Planned Templates

## Project Template

A standard structure for documenting hands-on projects.

Topics may include:

- Overview
- Objectives
- Environment
- Installation
- Configuration
- Challenges
- Troubleshooting
- Lessons Learned
- Future Improvements

---

## Lab Template

A reusable format for documenting labs from platforms such as:

- TryHackMe
- Hack The Box
- PortSwigger Web Security Academy
- Home Lab exercises

---

## Investigation Template

Used to document troubleshooting, incident investigations, and research.

Topics may include:

- Problem
- Investigation
- Findings
- Root Cause
- Resolution
- Lessons Learned

---

## Write-Up Template

Used for documenting completed rooms, challenges, or technical walkthroughs while respecting platform rules and avoiding spoilers when appropriate.

---

# Purpose

Templates help keep documentation:

- Consistent
- Organized
- Easy to review
- Easy to maintain
- Professional

As this repository grows, additional templates will be added for new projects and learning activities.
