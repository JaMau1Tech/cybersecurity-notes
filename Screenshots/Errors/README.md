# Error Screenshots

This folder contains screenshots of errors encountered during projects, labs, and troubleshooting exercises.

---

# Purpose

These screenshots help document:

- Error messages
- Installation failures
- Configuration issues
- Command-line mistakes
- Troubleshooting progress
- Successful resolutions

---

# Examples

```text
virtualbox-safe-graphics-error.png
wsl-feature-name-unknown.png
linux-directory-not-found.png
permission-denied-error.png
```

---

# Why This Matters

Documenting errors helps:

- Improve troubleshooting skills
- Build an incident history
- Identify recurring problems
- Develop better documentation habits
- Demonstrate problem-solving abilities
