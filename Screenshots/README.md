# Screenshots

This folder contains screenshots collected throughout my cybersecurity learning journey.

Screenshots are used to:

- Document progress
- Capture troubleshooting steps
- Save important terminal output
- Record lab activities
- Provide evidence of hands-on practice

---

# Folder Organization

```text
Screenshots/
├── Linux/
├── Networking/
├── TryHackMe/
├── WSL/
├── Errors/
└── Tools/
```

---

# Naming Convention

Use descriptive filenames:

```text
linux-first-login.png
nmap-first-scan.png
tryhackme-linux-fundamentals-complete.png
wsl-install-success.png
```

Avoid names such as:

```text
Screenshot 2025-07-06.png
IMG_1234.png
Untitled.png
```

---

# Purpose

Maintaining screenshots helps:

- Improve documentation skills
- Track learning progress
- Create a professional portfolio
- Make troubleshooting easier
- Provide evidence of hands-on experience

---

# Lesson Learned

Good documentation is an important skill in information technology and cybersecurity. Screenshots often explain problems and solutions better than text alone.
