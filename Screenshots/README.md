# Screenshots

This folder contains screenshots documenting my cybersecurity learning journey.

Screenshots provide visual evidence of projects, labs, troubleshooting, and technical concepts learned throughout my hands-on practice.

---

# Folder Structure

```text
Screenshots/
├── Errors/
├── Linux/
├── Networking/
├── Tools/
└── TryHackMe/
```

---

# Purpose

Screenshots are used to:

- Document project progress
- Capture errors and troubleshooting steps
- Record successful configurations
- Support written documentation
- Provide visual references for future review

---

# Naming Convention

Use descriptive lowercase filenames separated by hyphens.

Examples:

```text
linux-whoami-command.png
virtualbox-display-settings.png
wsl-install-success.png
nmap-scan-results.png
tryhackme-linux-find-command.png
```

Avoid generic names such as:

```text
Screenshot 2026-07-08.png
IMG_1234.png
Untitled.png
```

---

# Documentation Goal

Every major project, lab, or troubleshooting session should include screenshots that support the written documentation.
