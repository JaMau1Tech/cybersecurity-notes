# WSL (Windows Subsystem for Linux) Notes

## Goal
Learn Linux fundamentals without using a virtual machine.

## Why WSL?
- Lightweight
- Runs directly on Windows
- Great for command-line practice
- Useful for scripting and cybersecurity work

## Installation Steps
1. Open PowerShell as Administrator.
2. Run:

```powershell
wsl --install
```

3. Restart computer if prompted.
4. Launch Ubuntu.
5. Create a Linux username and password.

## Lessons Learned
- WSL provides a Linux environment inside Windows.
- Linux commands can be practiced without a VM.
- Documentation makes troubleshooting easier.
