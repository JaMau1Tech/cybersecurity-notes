# Investigation 01 - Windows System Recon

## Objective

Perform basic Windows system reconnaissance using built-in command-line tools to identify user information, system configuration, network settings, file structure, and storage layout.

---

## Skills Practiced

- Windows Command Prompt
- System Enumeration
- Network Enumeration
- File System Navigation
- Disk Enumeration
- Basic Incident Investigation

---

## Commands Used

```cmd
whoami
hostname
systeminfo
ipconfig
dir
diskpart
list volume
```

---

## Investigation Summary

This investigation focused on gathering information from a Windows workstation using native command-line tools.

Information collected included:

- Current logged-in user
- Computer hostname
- Windows version
- Hardware information
- Network configuration
- Directory contents
- Disk partitions and volumes
