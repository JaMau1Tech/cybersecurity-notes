# Screenshots

This directory contains screenshots captured during **Investigation 01 – Website Traffic Analysis**.

The screenshots document each stage of the investigation, from capturing live traffic to analyzing DNS, TCP, and TLS communications using Wireshark.

---

## Screenshot Naming Convention

```text
wireshark-home-screen
wireshark-capture-started
wireshark-capture-complete

```

---

## Purpose

These screenshots provide visual evidence of:

- Live packet capture
- DNS queries and responses
- TCP three-way handshake
- TLS handshake
- HTTPS encrypted communication

They support the investigation notes and demonstrate practical network traffic analysis using Wireshark.
