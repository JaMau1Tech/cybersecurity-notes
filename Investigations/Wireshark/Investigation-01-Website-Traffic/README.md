# Investigation 01 - Website Traffic Analysis

## Overview

This investigation documents the process of capturing and analyzing network traffic using Wireshark.

The objective was to understand what occurs behind the scenes when visiting websites by observing DNS lookups, TCP connection establishment, TLS negotiation, and encrypted HTTPS traffic.

---

## Objectives

- Capture live network traffic
- Identify DNS queries and responses
- Observe the TCP three-way handshake
- Analyze the TLS handshake
- Understand encrypted HTTPS communication
