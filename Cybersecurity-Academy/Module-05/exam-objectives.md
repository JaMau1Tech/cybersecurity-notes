# Purpose

This assessment validates the knowledge and practical skills gained from completing **Module 05 – Network Fundamentals** in the TryHackMe Pre Security learning path.

The objective is to demonstrate understanding of core networking concepts before progressing to **Module 06 – How the Web Works**.

The assessment focuses on comprehension, application, troubleshooting, and practical reasoning rather than memorization.

---

# Learning Objectives

After completing this assessment, the learner should be able to:

- Explain the purpose of computer networks.
- Differentiate between LANs, WANs, intranets, and the Internet.
- Explain the OSI Model and the responsibilities of each layer.
- Describe TCP/IP communication.
- Explain the TCP Three-Way Handshake.
- Compare TCP and UDP.
- Explain how packets and frames move across a network.
- Describe common networking protocols and their purposes.
- Explain how ports are used by applications.
- Identify common well-known ports.
- Explain how routers and switches operate.
- Differentiate Layer 2 and Layer 3 switching.
- Explain VLAN segmentation.
- Describe firewall functionality.
- Differentiate stateful and stateless firewalls.
- Explain VPN technologies and their purpose.
- Explain routing and packet forwarding.
- Apply networking knowledge to troubleshooting scenarios.
