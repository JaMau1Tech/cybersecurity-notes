# Root & Repository Cybersecurity Academy

## Overview

The **Root & Repository Cybersecurity Academy** is my personal knowledge validation and skill development program.

While TryHackMe provides structured learning paths and hands-on labs, this academy exists to ensure I **understand, retain, and can apply** what I have learned.

Instead of simply completing rooms, I challenge myself with exams, practical exercises, investigation scenarios, mock interviews, and capstone projects to reinforce my knowledge and build confidence before pursuing a career in IT and cybersecurity.

---

# Goals

- Reinforce knowledge learned through TryHackMe
- Improve long-term knowledge retention
- Practice explaining technical concepts
- Develop problem-solving skills
- Prepare for technical interviews
- Build confidence before entering the IT industry
- Create additional portfolio projects demonstrating independent learning

# Academy Components

## Knowledge Exams

Written exams designed to evaluate understanding of concepts learned throughout each module.

These exams include:

- Multiple Choice
- True / False
- Fill in the Blank
- Matching
- Short Answer
- Scenario-Based Questions

---

## Practical Exercises

Hands-on challenges requiring the application of knowledge rather than simple memorization.

Examples include:

- Binary conversions
- Hexadecimal conversions
- SQL query writing
- Python code analysis
- JavaScript code analysis
- Linux command usage
- Windows administration tasks

---

## Investigation Scenarios

Small investigation exercises that simulate real-world IT and cybersecurity situations.

Examples include:

- Windows Event Log analysis
- Network traffic investigation
- SQL database review
- Log analysis
- Browser artifact review
- Suspicious activity investigations

---

## Mock Interviews

Technical interview practice designed to improve confidence and communication skills.

Topics include:

- IT Fundamentals
- Networking
- Windows
- Linux
- Programming
- Databases
- Cybersecurity Concepts
- Troubleshooting

---

## Capstone Challenges

Larger projects combining multiple skills into realistic cybersecurity scenarios.

Examples may include:

- Home Lab Projects
- Incident Response Investigations
- Network Analysis
- Security Assessments
- Documentation Projects
