# Overview

This project documents my completion of Cisco Networking Academy's **Getting Started with Cisco Packet Tracer** course.

The course introduced Cisco Packet Tracer, a network simulation platform used to design, configure, and troubleshoot computer networks without requiring physical networking equipment.

Throughout the course, I learned how to build simple network topologies, configure end devices, understand logical and physical network layouts, and verify network connectivity using standard networking tools.

# Commands Used

## View Network Configuration

```text
ipconfig /all
```

Displays:

- IPv4 Address
- Subnet Mask
- Default Gateway
- DNS Server
- DHCP Information

---

## Test Connectivity

```text
ping cisco.srv
```

Verifies successful communication between the client and server.
