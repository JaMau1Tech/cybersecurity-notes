# Home Lab

## Overview

Welcome to my Home Lab portfolio.

This repository documents the design, implementation, and maintenance of a multi-project Home Lab built to develop practical skills in systems administration, networking, virtualization, and cybersecurity. Each project focuses on real-world technologies and is documented with technical notes, architecture documentation, and a professional project summary.

The Home Lab is designed to simulate enterprise environments while reinforcing hands-on learning and troubleshooting skills.
