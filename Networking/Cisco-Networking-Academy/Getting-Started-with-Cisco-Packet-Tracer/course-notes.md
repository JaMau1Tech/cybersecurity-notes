# Course Overview

This course introduced Cisco Packet Tracer, Cisco Networking Academy's network simulation and visualization software. Packet Tracer allows users to build, configure, and troubleshoot computer networks in a virtual environment without requiring physical networking equipment.

The course focused on learning the Packet Tracer interface, deploying networking devices, building simple networks, configuring end devices, and verifying connectivity using basic networking commands.

# Concepts Learned

## Cisco Packet Tracer

Packet Tracer is a network simulation platform that allows users to create virtual computer networks using Cisco routers, switches, servers, computers, wireless devices, and other networking equipment.

It provides a safe environment to practice networking concepts without the need for physical hardware.

---

## Logical Mode

Logical Mode focuses on how devices communicate with one another.

It emphasizes:

- Network topology
- Device connectivity
- Configuration
- Packet flow

---

## Physical Mode

Physical Mode represents the real-world placement of networking equipment.

Examples include:

- Wiring closets
- Office buildings
- Equipment racks
- Network cabling
- Physical device locations

## Networking Commands Practiced

### Display IP Configuration

```text
ipconfig /all
```

Displays:

- IPv4 Address
- Subnet Mask
- Default Gateway
- DNS Server
- DHCP Information

---

### Test Connectivity

```text
ping cisco.srv
```

Verifies communication between the client and server.
