# Module 07 – Attacks and Defenses

This module introduces the core principles of cybersecurity by exploring both offensive and defensive security. It explains what cybersecurity protects, how cryptography secures information, how ethical hackers identify weaknesses before attackers do, and how defenders protect systems, networks, and users through layered security controls.

By completing this module, I gained a foundational understanding of how attackers operate, how defenders respond, and how both perspectives work together to strengthen an organization's overall security posture.

# Module Summary

Module 07 combined the foundational concepts of cybersecurity into a practical security mindset. Rather than focusing solely on attackers or defenders, this module demonstrated how both perspectives complement one another.

Beginning with the CIA Triad, I learned what cybersecurity is ultimately designed to protect. I then explored cryptography and how encryption preserves confidentiality and integrity. From there, I shifted into the offensive security mindset by discovering hidden resources and performing password auditing using industry-standard tools such as Gobuster and Hydra. Finally, I learned how defenders identify assets, prioritize risks, implement layered security controls, and continuously improve an organization's security posture.

This module serves as the bridge between foundational IT knowledge and practical cybersecurity operations.
