# Overview

The **Cryptography Concepts** room introduces the fundamental principles of cryptography and explains how encryption protects digital information. It covers plaintext, ciphertext, keys, algorithms, symmetric encryption, asymmetric encryption, and how both methods work together to secure modern Internet communications such as HTTPS.

A hands-on Caesar Cipher exercise reinforces the concepts by demonstrating how encryption and decryption work in practice.

---

# Learning Objectives

- Understand the purpose of cryptography
- Explain plaintext and ciphertext
- Understand encryption and decryption
- Learn the role of keys and algorithms
- Understand symmetric encryption
- Understand asymmetric encryption
- Explain the key distribution problem
- Understand how HTTPS uses both encryption methods

# Key Concepts Learned

- Cryptography protects confidentiality and integrity.
- Plaintext is readable information.
- Ciphertext is encrypted information.
- Algorithms define how encryption works.
- Keys control encryption and decryption.
- Symmetric encryption uses one shared secret key.
- Asymmetric encryption uses public and private keys.
- HTTPS combines both encryption methods to create secure communications.
- Certificates verify the identity of websites.
