## SQL Query Process

The recap reviewed how to construct a filtered and sorted SQL query.

### Query Syntax

```sql
SELECT columns
FROM table
WHERE condition
ORDER BY column;
```

### Clause Purposes

- `SELECT` specifies which columns to retrieve.
- `FROM` identifies the table containing the data.
- `WHERE` filters records.
- `ORDER BY` sorts the filtered results.

## JavaScript Program Flow

The recap reviewed the logical order of the JavaScript number-guessing game.

1. Generate a random secret number using `Math.random()`.
2. Display instructions using `console.log()`.
3. Prompt the user using the readline question method.
4. Compare the guess to the secret using conditional statements.
