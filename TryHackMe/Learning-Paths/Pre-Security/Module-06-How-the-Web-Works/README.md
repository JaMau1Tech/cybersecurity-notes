# Module 06 – How The Web Works

## Overview

This module introduces the core technologies that make the World Wide Web function. It explores how browsers communicate with web servers, how domain names are translated into IP addresses, how websites are built, and how supporting infrastructure improves performance, scalability, and security.

By completing this module, I gained a solid understanding of the complete lifecycle of a web request, providing a strong foundation for future web application security and penetration testing topics.

---

# Module Objectives

- Understand the purpose of DNS
- Learn how HTTP and HTTPS work
- Understand website architecture
- Learn the difference between frontend and backend technologies
- Understand supporting web infrastructure
- Build a foundation for web security concepts
