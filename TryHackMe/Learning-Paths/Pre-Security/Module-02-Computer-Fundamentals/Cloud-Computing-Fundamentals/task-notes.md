## Key Concepts Learned

### Cloud Computing

Cloud computing is the delivery of computing services over the internet.

Examples of services include:

- Virtual machines
- Storage
- Databases
- Networking
- Applications

---

## Benefits of Cloud Computing

- Scalability
- High availability
- Reduced hardware costs
- On-demand resources
- Global accessibility
- Pay-as-you-go pricing

---

## Cloud Service Models

### Infrastructure as a Service (IaaS)

Provides virtualized hardware resources.

Examples:

- Amazon EC2
- Microsoft Azure Virtual Machines

Responsibilities:

- User manages operating systems and applications.
- Cloud provider manages hardware.

---

### Platform as a Service (PaaS)

Provides a platform for developing applications.

Examples:

- Heroku
- Google App Engine

Responsibilities:

- User manages applications.
- Cloud provider manages infrastructure.

---

### Software as a Service (SaaS)

Provides complete applications over the internet.

Examples:

- Gmail
- Microsoft 365
- Netflix

Responsibilities:

- Cloud provider manages everything.

---

## Cloud Instances

Cloud providers allow users to create virtual machines called instances.

Common actions include:

- Creating instances
- Starting instances
- Stopping instances
- Deleting instances
- Selecting deployment regions

---

## Cloud Billing

Cloud services typically use a pay-as-you-go model.

Important concepts:

- Running resources cost money.
- Stopped resources usually reduce costs.
- Monitoring usage helps avoid unnecessary expenses.

---

## Cloud Regions

Cloud providers operate data centers around the world.

Example:

```text
us-east-1
```

Choosing a region can affect:

- Performance
- Availability
- Cost
- Compliance requirements

---

## Cybersecurity Relevance

Cloud computing is important in cybersecurity because professionals regularly work with:

- Cloud-hosted applications
- Virtual machines
- Cloud security monitoring
- Identity and access management
- Cloud infrastructure security
