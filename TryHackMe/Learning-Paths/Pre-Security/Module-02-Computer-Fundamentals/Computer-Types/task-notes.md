## Key Concepts Learned

### Desktop Computer

A stationary computer designed for performance, upgradeability, and extended use.

Characteristics:

- Larger hardware components
- Better cooling
- Easy to upgrade
- Used in homes, schools, and offices

---

### Laptop Computer

A portable computer designed for mobility and convenience.

Characteristics:

- Battery powered
- Lightweight
- Limited upgradeability
- Compact hardware design

---

### Server

A computer that provides services, resources, or data to other computers on a network.

Examples:

- Web servers
- File servers
- Database servers
- Mail servers

Characteristics:

- High uptime
- Redundant hardware
- Supports multiple users

---

### Embedded System

A computer designed to perform a specific task.

Examples:

- Washing machines
- Microwaves
- Traffic lights
- Smart TVs

---

### Internet of Things (IoT)

Devices that connect to a network and communicate over the internet.

Examples:

- Smart thermostats
- Smart watches
- Security cameras
- Smart speakers
