## Overview

This section introduced the fundamental hardware components that make up a computer system and explained how these components work together to allow a computer to function.

## Learning Objectives

- Identify major computer components
- Understand the purpose of each component
- Learn how hardware components interact
- Understand the basic boot process
- Recognize the importance of system resources

---

## Key Concepts Learned

### CPU (Central Processing Unit)

The brain of the computer.

Responsibilities:

- Executes instructions
- Performs calculations
- Processes data
- Coordinates system operations

---

### RAM (Random Access Memory)

Temporary memory used by the operating system and applications.

Characteristics:

- Fast access speeds
- Volatile memory
- Data is lost when power is removed

---

### Storage

Long-term storage for:

- Operating systems
- Programs
- User files

Examples:

- SSD
- HDD

---

### Motherboard

The primary circuit board that connects all hardware components.

---

### GPU (Graphics Processing Unit)

Responsible for processing graphical information and displaying images.

---

### Power Supply Unit (PSU)

Provides electrical power to the computer's components.

---

### Network Adapter

Allows the computer to communicate with networks and the internet.

---

## Boot Process

The basic boot sequence:

1. Power button pressed
2. UEFI/BIOS starts
3. POST (Power-On Self-Test)
4. Boot device selected
5. Bootloader starts
6. Operating system loads
