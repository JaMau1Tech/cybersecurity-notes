## Key Concepts Learned

### Virtualization

The process of creating virtual versions of computing resources, such as:

- Operating systems
- Servers
- Storage
- Networks

Virtualization allows multiple systems to run on a single physical machine.

---

### Hypervisor

Software that creates and manages virtual machines.

Examples:

- Oracle VirtualBox
- VMware
- Hyper-V

Responsibilities:

- Allocating resources
- Managing virtual hardware
- Running multiple operating systems

---

### Virtual Machine (VM)

A software-based computer that behaves like a physical computer.

Characteristics:

- Has its own operating system
- Has virtual hardware
- Runs independently from other virtual machines

Examples:

- Ubuntu VM
- Windows Server VM
- Kali Linux VM

---

### Container

A lightweight environment that shares the host operating system.

Examples:

- Docker containers
- Kubernetes pods

Characteristics:

- Faster startup
- Lower resource usage
- Shares the host kernel

---

## Virtual Machines vs Containers

| Virtual Machine | Container |
|-----------------|------------|
| Includes full operating system | Shares host operating system |
| Higher resource usage | Lower resource usage |
| Slower startup | Faster startup |
| Greater isolation | Lightweight deployment |

- Testing environments
- Cloud computing

- Build home labs
- Test tools safely
- Practice penetration testing
- Analyze malware
- Create isolated environments
