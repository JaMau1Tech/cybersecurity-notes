## Overview

This section introduced the fundamentals of defensive security and the role security professionals play in protecting systems, networks, and users from cyber threats.

Defensive security focuses on detecting, investigating, and responding to attacks while improving an organization's security posture.

---

## Learning Objectives

- Understand the purpose of defensive security
- Learn the responsibilities of security teams
- Understand the role of Security Operations Centers (SOCs)
- Learn how analysts investigate attacks
- Understand basic incident response and mitigation

---

## Key Concepts Learned

### Defensive Security

The practice of protecting systems and networks from cyber threats through monitoring, detection, investigation, and response.

### Security Operations Center (SOC)

A centralized team responsible for monitoring and responding to security incidents.

### Incident Response

The process of identifying, investigating, containing, and recovering from security incidents.

### Indicators of Compromise (IOCs)

Evidence that may indicate a system has been compromised.

Examples:

- Malicious IP addresses
- Suspicious URLs
- Malware files
- Unusual network traffic
