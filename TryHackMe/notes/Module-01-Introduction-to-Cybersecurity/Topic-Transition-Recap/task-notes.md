## Key Concepts Learned

### Offensive Security

Security professionals who identify vulnerabilities by thinking like attackers.

### Defensive Security

Security professionals who monitor, investigate, and protect systems from cyber threats.

### Security Operations Center (SOC)

A team responsible for monitoring and responding to security incidents.

### Enumeration

The process of gathering information about a target system.

### Incident Response

The process of detecting, investigating, containing, and recovering from security incidents.
