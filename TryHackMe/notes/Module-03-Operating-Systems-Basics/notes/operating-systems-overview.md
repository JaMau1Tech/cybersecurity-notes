# Operating Systems Overview

## What is an Operating System?

An operating system (OS) is software that manages computer hardware and software resources while providing services for applications and users.

Examples:

- Windows
- Linux
- macOS
- Android
- iOS

---

## Main Responsibilities

- Managing hardware resources
- Managing memory
- Managing files and storage
- Running applications
- Managing users and permissions
- Providing user interfaces

---

## Types of User Interfaces

### Graphical User Interface (GUI)

Uses windows, icons, and menus.

Examples:
- Windows
- Ubuntu Desktop

### Command-Line Interface (CLI)

Uses text commands.

Examples:
- Linux Terminal
- PowerShell
- Command Prompt

---

## Why Operating Systems Matter in Cybersecurity

Understanding operating systems helps with:

- System administration
- Incident response
- Digital forensics
- Penetration testing
- Malware analysis
- Security hardening
