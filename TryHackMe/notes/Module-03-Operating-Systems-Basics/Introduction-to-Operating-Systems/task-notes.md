# Introduction to Operating Systems

## Status

✅ Completed

## Overview

This section introduced the fundamentals of operating systems, including their purpose, responsibilities, different operating system families, and how users interact with systems through graphical and command-line interfaces.

The room also explored file systems, user directories, and basic system information using a Linux virtual machine.

---

## Learning Objectives

- Understand the purpose of an operating system
- Explain the responsibilities of an operating system
- Identify different operating system types and families
- Understand the relationship between hardware, the operating system, applications, and users
- Differentiate between graphical and command-line interfaces
- Explore file systems and user directories
- Gather basic system information from a Linux system
