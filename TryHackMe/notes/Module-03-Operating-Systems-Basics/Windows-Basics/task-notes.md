## Overview

This section introduced application management, system configuration, and built-in security features within Windows. The room covered updating and installing applications, exploring Windows Settings and Control Panel, using Task Manager to monitor system activity, and performing security scans using Windows Security and Windows Defender.

---

## Learning Objectives

- Understand Windows Update and application updates
- Install and uninstall applications
- Explore Windows Settings and Control Panel
- Use Task Manager to monitor system activity
- Understand Windows Security and Windows Defender
- Perform a custom malware scan
