## Overview

This room introduced how computers represent colors and numbers using binary and hexadecimal. It explained how bits and bytes are used to store data, how RGB colors are represented digitally, and why hexadecimal is commonly used to simplify binary values.

---

## Learning Objectives

- Understand how computers represent colors
- Learn binary number representation
- Learn hexadecimal representation
- Understand bits and bytes
- Convert between binary, decimal, and hexadecimal

---

## Key Concepts Learned

- Computers use the binary (base-2) number system.
- A bit stores one of two values: 0 or 1.
- Eight bits equal one byte.
- RGB colors are represented using 24 bits (3 bytes).
- Each RGB value ranges from 0–255.
- Four bits equal one hexadecimal digit.
- Hexadecimal simplifies reading and writing binary values.
- Binary, decimal, and hexadecimal all represent the same values using different bases.
