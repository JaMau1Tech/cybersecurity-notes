# Python: Simple Demo

## Room Information

- **Room:** Python: Simple Demo
- **Module:** Module 04 – Software Basics
- **Difficulty:** Easy

---

## Objective

Learn the fundamentals of Python programming by creating a simple "Guess the Number" game while understanding variables, functions, conditional statements, and loops.

---

## Topics Covered

- Python Basics
- Variables
- Functions
- User Input
- Random Number Generation
- Conditional Statements
- While Loops
- Basic Program Flow

---

### Variables

- `secret`

- `tries`

### Functions

- `print()`
- `input()`
- `int()`
- `random.randint()`

---

### Conditional Statements

```python
if
elif
else
```

Used to compare the user's guess with the secret number.

---

### Loops

```python
while
```

Repeatedly executes code until the correct number is guessed.
