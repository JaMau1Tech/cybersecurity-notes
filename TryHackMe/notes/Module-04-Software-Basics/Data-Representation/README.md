## Key Concepts

### Computer Colors

- Computers combine Red, Green, and Blue (RGB) values to create colors.
- Each color channel uses one byte (8 bits).
- 24-bit color provides over 16 million possible colors.

### Binary

- Computers use only two states:
  - 0
  - 1

Each binary digit is called a **bit**.

### Hexadecimal

Hexadecimal provides a compact way to represent binary.

Example:

```
11111111 = FF
```

Every hexadecimal digit represents four binary bits.
