## Key Concepts Learned

- DNS translates domain names into IP addresses.
- HTTP and HTTPS allow browsers and web servers to communicate.
- HTML structures webpages while JavaScript provides interactivity.
- Backend servers generate dynamic content that browsers render as HTML.
- Load balancers distribute traffic and improve availability.
- CDNs improve website performance by serving static content from geographically distributed servers.
- Databases store application and user data.
- WAFs help protect web applications from common attacks.
- A complete web request involves DNS resolution, HTTP communication, server-side processing, and browser rendering.
