# Job Search Checklist

Use this checklist before submitting every Information Technology or Cybersecurity job application.

---

# Before Applying

## Company Research

- [ ] Reviewed company website
- [ ] Reviewed mission and values
- [ ] Reviewed job description
- [ ] Identified required technical skills
- [ ] Identified preferred qualifications
- [ ] Reviewed recent company news

---

## Resume

- [ ] Correct resume selected
  - [ ] Resume-IT
  - [ ] Resume-Cybersecurity
- [ ] Resume updated
- [ ] Resume exported as PDF
- [ ] Contact information verified
- [ ] GitHub link verified
- [ ] LinkedIn link verified
- [ ] No spelling or grammar errors

---

## Cover Letter

- [ ] Correct template selected
- [ ] Company name updated
- [ ] Hiring manager updated (if available)
- [ ] Job title updated
- [ ] Opening customized
- [ ] Relevant skills emphasized
- [ ] Closing customized
- [ ] Proofread

---

## GitHub Portfolio

- [ ] Latest projects pushed
- [ ] Documentation updated
- [ ] README files current
- [ ] Screenshots uploaded
- [ ] Repository links working

---

## LinkedIn

- [ ] Profile updated
- [ ] Featured section current
- [ ] Skills updated
- [ ] Certifications updated
- [ ] Recent projects added

---

# Application Submission

- [ ] Resume uploaded
- [ ] Cover letter uploaded
- [ ] Application reviewed
- [ ] Application submitted
- [ ] Confirmation email received

---

# After Applying

- [ ] Added application to applications.md
- [ ] Added follow-up reminder
- [ ] Saved job description
- [ ] Saved company information
- [ ] Scheduled follow-up (5–7 business days if appropriate)
- [ ] Connected with recruiter or hiring manager on LinkedIn (if appropriate)

---

# Interview Preparation (If Invited)

- [ ] Company research completed
- [ ] Reviewed STAR stories
- [ ] Reviewed common interview questions
- [ ] Reviewed technical interview questions
- [ ] Prepared questions for interviewer
- [ ] Reviewed GitHub portfolio
- [ ] Reviewed resume

---

# After the Interview

- [ ] Sent thank-you email within 24 hours
- [ ] Updated interview-log.md
- [ ] Updated follow-ups.md
- [ ] Recorded lessons learned
- [ ] Continued preparing for next interview
