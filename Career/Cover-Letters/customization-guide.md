# Cover Letter Customization Guide

This guide outlines the steps I should follow before submitting any job application to ensure every cover letter is tailored to the employer and position.

---

# Step 1: Research the Company

Before writing the cover letter, identify:

- Company name
- Industry
- Mission and values
- Products or services
- Recent news or achievements
- Company culture

---

# Step 2: Review the Job Description

Identify:

- Job title
- Required technical skills
- Preferred qualifications
- Responsibilities
- Keywords used throughout the posting

Highlight keywords that should naturally appear in the cover letter.

---

# Step 3: Choose the Correct Template

Use:

- Cover-Letter-IT.md
- Cover-Letter-Cybersecurity.md

If neither fits perfectly, begin with Cover-Letter-Master.md.

---

# Step 4: Customize the Opening

Update:

- Company name
- Job title
- Reason for applying
- Why I am interested in this organization

Avoid generic openings.

---

# Step 5: Match My Experience

Select the experiences that best match the job posting.

Examples:

- Customer Service
- Technical Documentation
- Networking
- Linux
- Windows
- GitHub Portfolio
- TryHackMe
- Troubleshooting
- Security Fundamentals

Focus on the experiences most relevant to the role.

---

# Step 6: Use Job Description Keywords

Naturally include keywords from the posting.

Examples:

- Technical Support
- Help Desk
- Windows
- Linux
- Networking
- Customer Service
- Active Directory
- Office 365
- Troubleshooting
- Cybersecurity
- Security Operations
- Documentation

Do not force keywords where they don't fit naturally.

---

# Step 7: Customize the Closing

Choose the most appropriate closing paragraph from:

- closing-paragraphs.md

---

# Step 8: Final Review

Verify:

- Company name is correct
- Job title is correct
- No spelling or grammar errors
- Resume version matches the position
- Cover letter matches the position
- GitHub portfolio is up to date
- Contact information is correct

---

# Goal

Every employer should feel that the cover letter was written specifically for their position while remaining truthful and accurately representing my experience and skills.
