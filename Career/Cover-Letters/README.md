# Cover Letters

This folder contains the source files used to create and maintain professional cover letters for Information Technology and Cybersecurity positions.

Rather than writing a new cover letter for every application, reusable templates and supporting documents are maintained here. Each application begins with the Master Cover Letter and is then customized for the specific company and position.

---

## Workflow

Cover-Letter-Master.md
↓

Cover-Letter-IT.md

OR

Cover-Letter-Cybersecurity.md

↓

Customize for Company

↓

Submit Application
