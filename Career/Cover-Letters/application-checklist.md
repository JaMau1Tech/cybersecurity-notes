# Job Application Checklist

Use this checklist before submitting every Information Technology or Cybersecurity job application.

---

# Resume

- [ ] Correct resume selected
  - [ ] Resume-IT
  - [ ] Resume-Cybersecurity
- [ ] Resume reviewed
- [ ] Resume exported as PDF
- [ ] No spelling or grammar errors
- [ ] Contact information is correct
- [ ] GitHub link works
- [ ] LinkedIn link works

---

# Cover Letter

- [ ] Correct template selected
- [ ] Company name updated
- [ ] Hiring manager updated (if available)
- [ ] Job title updated
- [ ] Opening customized
- [ ] Relevant experience emphasized
- [ ] Closing customized
- [ ] Proofread for spelling and grammar

---

# LinkedIn

- [ ] Profile updated
- [ ] Open to Work status reviewed
- [ ] Featured section current
- [ ] Latest projects added
- [ ] Skills updated
- [ ] Certifications updated

---

# GitHub Portfolio

- [ ] Latest projects pushed
- [ ] README files updated
- [ ] Documentation complete
- [ ] Screenshots uploaded
- [ ] Repository links verified

---

# Company Research

- [ ] Company researched
- [ ] Job description reviewed
- [ ] Required skills identified
- [ ] Company mission understood
- [ ] Notes recorded

---

# Application

- [ ] Resume uploaded
- [ ] Cover letter uploaded
- [ ] Application reviewed
- [ ] Submitted successfully
- [ ] Confirmation email received

---

# After Applying

- [ ] Added to Job Tracker
- [ ] Saved job description
- [ ] Saved company website
- [ ] Saved application date
- [ ] Scheduled follow-up reminder
- [ ] Connected with recruiter or hiring manager on LinkedIn (if appropriate)

---

# Final Question

Before clicking **Submit**, ask yourself:

> "If I were the hiring manager, would this application clearly demonstrate my professionalism, technical growth, and genuine interest in this position?"
