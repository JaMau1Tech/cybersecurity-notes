# Interview Checklist

Use this checklist before every Information Technology or Cybersecurity interview.

---

# Before the Interview

## Company Research

- [ ] Review company website
- [ ] Review mission and values
- [ ] Review products or services
- [ ] Review recent company news
- [ ] Review job description
- [ ] Review interviewer(s) if known

---

## Resume & Portfolio

- [ ] Review Resume
- [ ] Review GitHub Portfolio
- [ ] Review LinkedIn Profile
- [ ] Review recent technical projects
- [ ] Review TryHackMe documentation
- [ ] Review STAR stories

---

## Technical Review

- [ ] Operating Systems
- [ ] Networking
- [ ] Windows
- [ ] Linux
- [ ] Command Line
- [ ] Git & GitHub
- [ ] Troubleshooting
- [ ] Cybersecurity Fundamentals

---

## Interview Preparation

- [ ] Practice "Tell me about yourself"
- [ ] Review common interview questions
- [ ] Review behavioral questions
- [ ] Review technical questions
- [ ] Prepare questions for interviewer

---

# Virtual Interview Checklist

- [ ] Camera tested
- [ ] Microphone tested
- [ ] Internet connection verified
- [ ] Quiet workspace
- [ ] Professional background
- [ ] Phone on silent
- [ ] Meeting link tested

---

# In-Person Interview Checklist

- [ ] Professional attire
- [ ] Resume copies printed
- [ ] Notebook and pen
- [ ] Government-issued ID (if required)
- [ ] Directions confirmed
- [ ] Arrive 10–15 minutes early

---

# During the Interview

- [ ] Maintain eye contact
- [ ] Listen carefully
- [ ] Ask clarifying questions if needed
- [ ] Answer using the STAR method when appropriate
- [ ] Demonstrate enthusiasm
- [ ] Highlight continuous learning
- [ ] Discuss GitHub portfolio naturally
- [ ] Remain professional and confident

---

# After the Interview

- [ ] Thank the interviewer
- [ ] Record interview notes
- [ ] Send a thank-you email within 24 hours
- [ ] Update the Job Tracker
- [ ] Reflect on strengths and areas for improvement
- [ ] Follow up if appropriate
