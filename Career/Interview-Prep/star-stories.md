## Action

- Completed TryHackMe learning paths
- Built the Root & Repository Cybersecurity Academy
- Learned Git and GitHub
- Practiced networking, Linux, Windows, Python, JavaScript, and SQL

Developed a structured technical portfolio demonstrating continuous learning, organization, and commitment to a long-term career in Information Technology and Cybersecurity.
