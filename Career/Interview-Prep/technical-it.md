# Technical IT Interview Questions

This document contains common technical interview questions for Help Desk, IT Support, Desktop Support, and other entry-level Information Technology positions.

---

# What is an Operating System?

An operating system (OS) is software that manages a computer's hardware and software resources while providing a platform for applications to run.

Examples include:

- Windows
- Linux
- macOS

---

# What is the difference between hardware and software?

Hardware is the physical components of a computer.

Examples:

- CPU
- RAM
- Hard Drive
- Keyboard
- Monitor

Software consists of the programs and operating systems that run on the hardware.

Examples:

- Windows
- Microsoft Office
- Google Chrome

---

# What is an IP Address?

An IP (Internet Protocol) address uniquely identifies a device on a network so it can communicate with other devices.

There are two common types:

- Public IP Address
- Private IP Address

---

# What is a MAC Address?

A MAC (Media Access Control) address is a unique hardware address assigned to a network interface card (NIC).

Unlike IP addresses, MAC addresses are designed to uniquely identify physical network hardware.

---

# What does Ping do?

Ping uses ICMP (Internet Control Message Protocol) to test connectivity between devices.

It helps determine:

- Whether a device is reachable
- Network latency
- Packet loss

---

# What is DNS?

DNS (Domain Name System) translates domain names into IP addresses.

Example:

google.com

↓

142.250.xxx.xxx

Without DNS, users would need to remember IP addresses instead of website names.

---

# What is DHCP?

DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses and other network settings to devices joining a network.

---

# What is the Command Line?

The command line allows users to interact with an operating system by typing commands instead of using a graphical interface.

Examples:

Windows

- Command Prompt
- PowerShell

Linux

- Bash

---

# What is Git?

Git is a version control system used to track changes to files and projects over time.

Benefits include:

- Version history
- Collaboration
- Backup
- Rollback

---

# What is GitHub?

GitHub is a cloud platform used to host Git repositories.

It enables:

- Portfolio development
- Collaboration
- Documentation
- Version control

# Describe your troubleshooting process.

My troubleshooting process is:

1. Identify the problem.
2. Gather information.
3. Reproduce the issue if possible.
4. Research potential causes.
5. Test one solution at a time.
6. Verify the solution.
7. Document the findings and resolution.
