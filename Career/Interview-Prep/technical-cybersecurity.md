# Technical Cybersecurity Interview Questions

This document contains common technical interview questions for entry-level Cybersecurity, SOC Analyst, and Security Operations positions.

---

# What is Cybersecurity?

Cybersecurity is the practice of protecting systems, networks, applications, and data from unauthorized access, attacks, damage, or theft.

Its primary goals are to:

- Protect confidentiality
- Maintain integrity
- Ensure availability

---

# What is the CIA Triad?

The CIA Triad is the foundation of information security.

### Confidentiality

Ensure only authorized users can access information.

Examples:

- Passwords
- Encryption
- Multi-Factor Authentication (MFA)

### Integrity

Ensure data is accurate and has not been modified without authorization.

Examples:

- Hashing
- Digital Signatures

### Availability

Ensure systems and data remain accessible when needed.

Examples:

- Backups
- Redundant Systems
- Disaster Recovery

---

# What is Authentication?

Authentication is the process of verifying a user's identity before granting access.

Examples:

- Passwords
- MFA
- Biometrics
- Smart Cards

---

# What is Authorization?

Authorization determines what an authenticated user is allowed to access or do.

Authentication answers:

"Who are you?"

Authorization answers:

"What are you allowed to do?"

---

# What is Malware?

Malware is malicious software designed to damage, disrupt, or gain unauthorized access to systems.

Common types include:

- Viruses
- Worms
- Trojans
- Ransomware
- Spyware

---

# What is Phishing?

Phishing is a social engineering attack where attackers attempt to trick users into revealing sensitive information.

Examples:

- Fake emails
- Fake login pages
- Fraudulent text messages

---

# What is a Firewall?

A firewall monitors and filters incoming and outgoing network traffic based on security rules.

It helps prevent unauthorized access to systems and networks.

---

# What is a VPN?

A Virtual Private Network (VPN) creates an encrypted connection between a user and a remote network.

Benefits include:

- Privacy
- Secure remote access
- Data encryption

---

# What is a SOC?

A Security Operations Center (SOC) is a team responsible for monitoring, detecting, investigating, and responding to cybersecurity threats.

SOC analysts continuously monitor systems to identify suspicious activity.

---

# What is a SIEM?

A Security Information and Event Management (SIEM) platform collects and analyzes logs from multiple systems to help identify security events.

Examples include:

- Microsoft Sentinel
- Splunk
- IBM QRadar

---

# What is Incident Response?

Incident Response is the structured process of identifying, containing, investigating, eradicating, and recovering from cybersecurity incidents.

Typical phases include:

1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

---

# What is Defense in Depth?

Defense in Depth is a security strategy that uses multiple layers of protection instead of relying on a single security control.

Examples:

- Firewalls
- MFA
- Antivirus
- Network Segmentation
- Security Awareness Training

---

# Why are logs important?

Logs record system and user activity.

Security professionals use logs to:

- Detect suspicious behavior
- Investigate incidents
- Support forensic analysis
- Troubleshoot technical issues
