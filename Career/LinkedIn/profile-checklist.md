# LinkedIn Profile Checklist

Use this checklist before actively applying for Information Technology or Cybersecurity positions.

---

# Profile Photo

- [ ] Professional headshot
- [ ] Clean background
- [ ] Business casual attire
- [ ] Friendly and approachable appearance

---

# Banner Image

- [ ] Professional technology-themed banner
- [ ] Clean and uncluttered design
- [ ] Consistent with my personal brand

---

# Headline

- [ ] Updated using linkedin-headline.md
- [ ] Clearly reflects career goals
- [ ] Includes relevant technical keywords

---

# About Section

- [ ] Updated using about.md
- [ ] Free of spelling and grammar errors

# Education
