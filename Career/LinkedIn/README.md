# LinkedIn

This folder contains the source files used to build and maintain my professional LinkedIn profile.

Rather than editing my LinkedIn profile directly, all profile content is created and maintained here first using Markdown. This provides version control, consistency with my Resume workflow, and an organized history of professional updates.
